## 1) Название

### **D&D Character Manager**

Модуль управления персонажами для настольной ролевой игры **Dungeons & Dragons 5-й редакции**

---

## 2) Краткое описание

**D&D Character Manager** — это автономный программный модуль, разработанный на платформе **Unity**, который позволяет создавать, редактировать, отслеживать и развивать игровых персонажей в системе Dungeons & Dragons 5-й редакции. Модуль полностью автоматизирует расчеты по правилам D&D 5e, включая модификаторы характеристик, максимальное здоровье, класс брони, бонусы владения и систему опыта.

### Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| **Создание персонажей** | Выбор расы, класса и распределение характеристик |
| **Автоматические расчеты** | Модификаторы характеристик, максимальное HP, класс брони |
| **Управление здоровьем** | Нанесение урона, лечение, временные HP с визуальным отображением |
| **Система уровней** | Прогрессия от 1 до 20 уровня с автоматическим расчетом HP |
| **Система опыта** | Накопление опыта и автоматическое повышение уровня |
| **Сохранение данных** | Автоматическое сохранение в JSON формате |
| **Кроссплатформенность** | Поддержка Android и Windows |

## 3) Системные требования

### Для разработки (Unity Editor)

| Компонент | Минимальные требования | Рекомендуемые требования |
|:----------|:----------------------|:-------------------------|
| **ОС** | Windows 10 (64-bit) / macOS 10.15+ | Windows 11 / macOS 13+ |
| **Процессор** | Intel Core i5-2500K / AMD Ryzen 3 1200 | Intel Core i7-8700K / AMD Ryzen 5 3600 |
| **ОЗУ** | 8 ГБ | 16 ГБ |
| **Видеокарта** | DX10-совместимая с 2 ГБ VRAM | DX11-совместимая с 4 ГБ VRAM |
| **Дисковое пространство** | 5 ГБ | 10 ГБ |
| **Unity Version** | 6.3 LTS или выше |

### Для запуска готового приложения

| Платформа | Минимальные требования |
|:----------|:----------------------|
| **Android** | Android 7.0 (API 24) или выше, 2 ГБ RAM, 50 МБ свободного места |
| **Windows** | Windows 10 (64-bit), DirectX 11, 4 ГБ RAM |
